# UI 设计提示词速查卡

> 一页式快速参考，收藏备用

---

## 一、提示词六要素

| 要素 | 说明 | 关键问题 | 示例 |
|:---:|------|---------|------|
| **任务** | 做什么 | 动作是什么？ | 设计/生成/优化/分析/重构 |
| **对象** | 设计什么 | 设计哪个页面/组件？ | 登录页/首页/按钮/卡片/图标 |
| **风格** | 什么调性 | 视觉风格是什么？ | 简约/科技/温馨/高端/年轻 |
| **功能** | 包含什么 | 需要哪些功能元素？ | 搜索框/导航/表单/列表/轮播 |
| **约束** | 什么限制 | 有什么技术限制？ | 平台/尺寸/品牌色/适配/性能 |
| **输出** | 要什么结果 | 期望什么输出？ | 描述/规格/代码/对比方案 |

## 二、万能模板

### 基础模板

```
请为 [用户] 设计一个 [对象]

【风格】[风格词] + 主色 [#XXXXXX]
【功能】① ② ③ ④
【约束】平台: [iOS/Android/Web] | 尺寸: [WxH]
【输出】[描述/规格/代码]
```

### 完整模板

```
## 设计任务
为 [目标用户] 设计 [产品类型] 的 [页面名称]

## 用户场景
[谁] 在 [什么情况下] 希望 [完成什么目标]

## 视觉风格
- 风格关键词：[风格1]、[风格2]
- 主色：[#XXXXXX] - [用途说明]
- 辅助色：[#XXXXXX] - [用途说明]
- 参考：[参考产品/风格]

## 功能要求
- [ ] [功能1]
- [ ] [功能2]
- [ ] [功能3]

## 技术约束
- 平台：[iOS/Android/Web/响应式]
- 尺寸：[宽×高]
- 适配：[暗色模式/小屏/横屏]

## 输出要求
请提供：[具体输出内容]
```

## 十三、自检清单

### 提交前必检

- [ ] **六要素完整**：任务/对象/风格/功能/约束/输出都有
- [ ] **对象具体**：说明了什么App、什么页面
- [ ] **用户明确**：说明了目标用户和使用场景
- [ ] **风格可执行**：有风格词+具体色值，非模糊词
- [ ] **功能完整**：核心功能都列出了
- [ ] **约束清晰**：平台、尺寸、适配要求明确
- [ ] **输出明确**：说明要什么形式的输出
- [ ] **无矛盾**：各项要求之间不冲突
- [ ] **量级合理**：单次任务量不过大

### 结果不满意时排查

- [ ] 是否缺少关键信息？→ 补充具体要求
- [ ] 是否有模糊表达？→ 改用具体数值
- [ ] 是否要求矛盾？→ 明确优先级
- [ ] 是否一次要太多？→ 拆分步骤
- [ ] 是否缺少参考？→ 提供参考示例
