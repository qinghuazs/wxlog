# UI 设计词汇表

## 前言

在与 AI 沟通 UI 设计需求时，使用准确的设计术语能显著提升沟通效率和结果质量。本章整理了 UI 设计中常用的专业词汇，涵盖布局、颜色、组件、状态、交互等多个维度，是你的提示词工具箱。

## 四、状态词汇（State）

### 交互状态

| 中文 | 英文 | 说明 | 视觉变化 |
|-----|------|------|---------|
| 默认 | Default / Normal | 初始状态 | 基础样式 |
